
## WORLD HAPPINESS REPORT 2017 ANALYSIS

Using Exploratory Data Analysis (EDA) on World Happiness Report 2017 using, relationships and correlation between different factors are found.

### Objectives
  1. Load and clean the dataset
  2. Perform EDA
  3. To find correlations & relationships and visualise them
  4. Draw conclusions from the results

### Libraries Used
  1. NumPy
  2. Pandas
  3. Matplotlib
  4. Seaborn
  5. Sci-kit Learn
  6. Scipy

### Dataset Used
  World Happiness Report 2017: https://www.kaggle.com/datasets/unsdsn/world-happiness?select=2017.csv
